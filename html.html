Html

O que é Html?

A sigla Html significa Linguagem de Marcação de Hipertexto, uma das linguagens da maior partes dos aplicativos online páginas da internet. Um hipertexto é um texto usado para fazer referência a outros textos, enquanto uma linguagem de marcação é composta por uma série de marcações que dizem para os servidores da web qual é o estilo e a estrutura de um documento. Porém, ele não é considerado como uma linguagem de programação, já que ele não pode criar funcionalidades dinâmicas, mas, os usuários podem criar e estruturar seções, parágrafos e links usando elementos, tags e atributos.

Tags

  • Tag p: Esta tag é utilizada para definir um parágrafo de texto. É uma das tags mais básicas e amplamente usadas em HTML para estruturar e organizar o conteúdo textual de uma página web.

Tag ol: define uma lista ordenada, onde os itens são numerados automaticamente pelo navegador. É útil quando a ordem dos itens é importante. EX:

  1. primeiro item
  2. segundo item
  3. terceiro item

Tag ul: define uma lista não ordenada, onde os itens são apresentados com marcadores (bullets) em vez de números. É útil quando a ordem dos itens não é importante. EX:

  • primeiro item
  • segundo item
  • terceiro item

Tag Table: é usada para criar tabelas que organizam dados em linhas e colunas. Uma tabela em HTML é composta por várias tags auxiliares que definem a estrutura e o conteúdo da tabela.

table: Define o início e o fim da tabela.

tr (table row): Define uma linha na tabela.

th (table header): Define uma célula de cabeçalho em uma linha de cabeçalho.

td (table data): Define uma célula de dados em uma linha.

Tag img: em HTML é usada para incorporar imagens em uma página web. Ela é um elemento vazio, o que significa que não possui uma tag de fechamento e apenas contém atributos para definir a imagem a ser exibida e suas propriedades

Tag a: é usada para incorporar imagens em uma página web. Esta tag é essencial para adicionar elementos visuais às suas páginas, proporcionando uma maneira fácil de exibir imagens como parte do conteúdo.
