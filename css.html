CSS

O que é CSS?

A sigla CSS significa Cascading Style Sheets é uma linguagem usada para descrever a apresentação de um documento escrito em HTML ou XML. Em termos simples, CSS define como os elementos em uma página web devem ser exibidos, permitindo controlar a aparência visual de um site. Sendo usado para estilizar esse conteúdo (como cores, fontes, espaçamento, etc.). Isso permite uma clara separação entre o que é mostrado (conteúdo) e como é mostrado (estilo). O termo "cascading" refere-se a como as regras de estilo são aplicadas. Se houver conflitos entre várias regras de estilo, o CSS tem um sistema de prioridade para determinar quais regras aplicar.

Seletores básicos

CSS utiliza seletores para determinar quais elementos HTML serão estilizados, podendo criar estilos complexos e precisos para qualquer elemento em uma página web.

Propriedades

src: Define o caminho para o arquivo de imagem.

alt: Fornece um texto alternativo para a imagem caso ela não possa ser exibida. É importante para acessibilidade e SEO.

width: Define a largura da imagem em pixels ou em porcentagem. Pode ser usado para redimensionar a imagem.

height: Define a altura da imagem em pixels ou em porcentagem. Pode ser usado para redimensionar a imagem.

title: Oferece informações adicionais sobre a imagem. Esse texto é exibido quando o usuário passa o cursor sobre a imagem.

loading: Controla o comportamento de carregamento da imagem. Pode ser lazy (para carregamento preguiçoso, quando a imagem está fora da tela) ou eager (para carregamento imediato).

style: Permite adicionar estilos CSS diretamente à imagem.

class: Atribui uma ou mais classes CSS à imagem para estilização.

id: Atribui um identificador único à imagem para uso em JavaScript ou CSS.
